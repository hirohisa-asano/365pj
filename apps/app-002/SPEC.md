# 副業バレリスク診断 — SPEC

## 概要
副業の種類・年収・本業の状況を入力すると「会社にバレるリスク（0-100%）」と具体的な対策が返ってくる診断ツール。AI API不要のルールベース判定。

## ターゲットユーザー
- 会社員で副業をしている or 始めたい人（800万人超）
- 副業解禁されたが住民税や確定申告の仕組みがわからず不安な人

## 課題
- 副業が会社にバレないか不安
- 住民税の仕組み（特別徴収 vs 普通徴収）がわからない
- 確定申告が必要かの判断があいまい
- 「副業 バレる」で検索すると記事は大量にあるが、自分の状況に合わせた診断ツールがない

## 競合状況
- 副業税額シミュレーター（マネーフォワード等）はあるが「バレるか」の診断ツールは存在しない
- 記事・ブログは多数だがインタラクティブなツールはゼロ
- 競合なし

## 画面構成

### `/` — メインページ（1ページ完結・ステップフォーム → 結果）

```
ヘッダー
├── アプリ名 + キャッチコピー
│
├── ステップフォーム（3ステップ）
│   ├── Step 1: 本業情報
│   │   ├── 就業規則での副業規定（選択: 禁止 / 許可制 / 自由 / わからない）
│   │   └── 会社の規模（選択: 大企業 / 中小企業 / スタートアップ）
│   │
│   ├── Step 2: 副業情報
│   │   ├── 副業の種類（選択: フリーランス・業務委託 / アルバイト・パート / 投資・不動産 / ネット副業・アフィリエイト / コンテンツ販売）
│   │   ├── 年間副業収入見込み（スライダー: 0〜500万円）
│   │   └── 副業の公開度（選択: 完全匿名 / 実名で活動 / SNSで発信）
│   │
│   └── Step 3: 対策状況
│       ├── 確定申告の経験（選択: したことある / 初めて / 不要な額）
│       └── 住民税の納付方法（選択: 普通徴収（自分で納付）/ 特別徴収（給与天引き）/ わからない）
│
└── 結果エリア（診断後に表示）
    ├── バレリスクスコア（0-100%）大きな数字で表示
    ├── リスクレベル（🟢安全 / 🟡注意 / 🔴危険）
    ├── バレるルート解説（該当するものだけ表示）
    │   ├── 住民税ルート: 特別徴収で額が増え、経理が気づく
    │   ├── 社会保険ルート: アルバイトで社保加入すると通知が届く
    │   ├── SNS・密告ルート: 実名活動・同僚に話す
    │   └── マイナンバールート: 直接バレることはない（安心材料）
    ├── 対策チェックリスト（チェックボックス付き）
    │   例: □ 確定申告で住民税を「自分で納付」にする
    │   例: □ SNSでは匿名で活動する
    │   例: □ 同僚に副業の話をしない
    ├── 確定申告が必要かの判定（年20万円ルール解説）
    └── SNSシェアボタン（「副業バレリスク○%でした」）

フッター（寄付 + ポータル）
```

## 技術仕様
- **レンダリング**: SSG（全てクライアントサイドで計算。サーバー通信なし）
- **DB**: 不要
- **認証**: 不要
- **外部API**: 不要（ルールベース判定。税制ロジックはハードコード）
- **Supabase**: 不使用

## 機能要件
1. 3ステップのフォームで情報を入力できる
2. 入力内容からバレリスクスコア（0-100%）をルールベースで算出する
3. リスクレベルを3段階（安全 / 注意 / 危険）で表示する
4. バレるルートを該当するものだけ解説付きで表示する
5. 対策チェックリストをチェックボックス付きで表示する
6. 確定申告が必要かを判定・表示する
7. X（Twitter）/ LINEでシェアできるボタンを設置する
8. レスポンシブ対応（モバイルファースト）
9. ステップ間をスムーズに遷移できる（アニメーション付き）
10. 「もう一度診断する」ボタンでリセットできる

## スコア算出ロジック（ルールベース）

### ベーススコア（副業規定）
- 禁止: +40
- 許可制: +15
- 自由: +0
- わからない: +25

### 副業種類の加算
- アルバイト・パート: +20（社保ルートあり）
- フリーランス・業務委託: +10
- 投資・不動産: +5（副業扱いされにくい）
- ネット副業: +10
- コンテンツ販売: +10

### 収入による加算
- 20万円以下: +0
- 20〜100万円: +10
- 100〜300万円: +15
- 300万円超: +20

### 公開度による加算
- 完全匿名: +0
- 実名で活動: +15
- SNSで発信: +20

### 住民税対策による減算
- 普通徴収: -15
- 特別徴収: +0
- わからない: +10

### 会社規模による補正
- 大企業: +5（経理が厳密）
- 中小企業: +0
- スタートアップ: -5（副業に寛容な傾向）

### スコア = min(max(合計, 0), 100)

## デザインの方向性
- **安心感**: 白背景 + グリーン系（安全）/ レッド系（危険）
- **カジュアル**: 堅すぎない。副業を応援するトーン
- **数字が主役**: バレリスクの%を大きく表示
- **ステップフォーム**: プログレスバー付きで現在位置がわかる

## スコープ外
- ユーザー登録・ログイン
- 診断結果の保存
- 税理士・弁護士への相談機能
- 具体的な確定申告の作成支援
- 個別の自治体ごとの住民税率計算
