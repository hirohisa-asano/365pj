## Pass 3：30 → 10（最終選考）

基準：
- **今すぐ自分が作れる**（3日以内）
- **競合に対して明確な角度がある**
- **最初の1人が見えている**

| 順位 | アイデア | 決め手 |
| -- | ------- | ------ |
| 🥇1 | **LLMコスト計算** | AI開発者が全員困っている。競合が弱い。今すぐ使いたい。 |
| 🥇2 | **フリーランス確定申告** | 毎年の強い痛み。日本語特化で勝てる。 |
| 🥇3 | **コードレビューAI** | 毎日使う。チーム導入で課金スケールする。 |
| 🥇4 | **プライバシーポリシー生成** | 全スタートアップが必要。3分で作れる価値。 |
| 🥇5 | **補助金検索** | 取れるはずの金。付加価値が高い。競合少ない。 |
| 🥇6 | **AI壁打ち** | 深い対話。月額課金に自然につながる。 |
| 🥇7 | **英会話相手** | 巨大市場。差別化余地あり（AI特化・低価格）。 |
| 🥇8 | **炎上リスク診断** | B2B。SNS投稿前チェックとして企業が払う。 |
| 🥇9 | **転職偏差値** | バイラル入口 → 詳細診断で課金。 |
| 🥇10 | **ソクラテスと対話** | 唯一無二。哲学・思考好きのコア層が課金する。 |
